# Contributing to Project Explorer Plugin

Thank you for your interest in contributing!

## Development Setup

1. Clone this repository
2. Copy the plugin to your Hermes Agent plugins directory:
   ```bash
   cp -r project-explorer ~/.hermes/hermes-agent/plugins/
   ```

3. Start Hermes Agent and navigate to the Projects tab to test

## Code Style

- Python: Follow PEP 8, use Black for formatting
- JavaScript: Use ESLint with the recommended config

## Submitting Changes

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request
